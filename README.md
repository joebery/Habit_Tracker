# Habit Tracker (Android)

A simple, offline-first habit tracking Android app inspired by minimal habit trackers with visual streak tracking.

This project is a work in progress and is being built as part of my learning journey while I learn Kotlin, Jetpack Compose, and modern Android development.

---

## Project Status

Work in progress.

This app is actively being developed while I learn:
- Kotlin
- Jetpack Compose
- Room (SQLite)
- Android Widgets (Jetpack Glance)

Features, structure, and UI may change as the project evolves.

---

## Goal

The goal of this project is to:
- Learn modern Android development practices
- Build a clean and usable habit tracker
- Support home screen widgets
- Keep all data local and offline

This project serves both as a learning exercise and a portfolio piece.

---

## Planned Features

- Create and manage daily habits
- Mark habits as completed for the day
- Visual streak tracking
- Minimal, dark-themed user interface
- Home screen widget support
- Local storage using Room (no backend)

---

## Tech Stack

- Language: Kotlin
- UI: Jetpack Compose
- Architecture: MVVM
- Local Storage: Room (SQLite)
- Widgets: Jetpack Glance
- Minimum SDK: 26+
